# MEU

## サイト概要
ハンバーガーネット注文サイト

### サイトテーマ
ハンバーガーネット注文サイト（店舗受け取り）

### テーマを選んだ理由
自分の中でWEBプログラマーは企業のHP作成依頼のイメージが強く、実践的であると考えたため。
チーム開発では携わらなかった場所も学習しておきたかったため。

### ターゲットユーザ
商品の待ち時間を減らしたい方

### 主な利用シーン
商品を先に注文できる
